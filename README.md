# 📄 RAG PDF Search

A Retrieval-Augmented Generation (RAG) based application that allows users to query PDF documents using natural language.

## 🚀 Features
- Upload and process PDFs
- Text chunking and embeddings
- Semantic search
- AI-powered question answering
- Fast and accurate responses

## 🛠️ Tech Stack
- Python
- LangChain
- FAISS / ChromaDB
- OpenAI / Gemini API
- Streamlit

## 📂 Project Structure
rag_pdf_search/
│── app.py  
│── utils.py  
│── data/  
│── embeddings/  
│── requirements.txt  

## ▶️ How to Run
pip install -r requirements.txt  
streamlit run app.py  

## 📌 Use Cases
- Research papers
- Legal documents
- Knowledge base search
